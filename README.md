# Growth- is a web application for all young generation in Kyrgyzstan, and for everyone who wants to improve themselves, prepare for ORT, School Subject Olympiads. This web application is especially for school students of remote part of Kyrgyzstan. The main goal of this web application is education accessibility and quality, resources in most regions of our country is not same as in cities or towns. We need to increase the quality of education in Kyrgyzstan. And we wanted to solve this problem somehow, and make some improvements in this area. This system will be designed to maximize the productivity of school students and at the same time to increase the education quality by providing features/tools which help them develop themselves, learn something new, prepare for entry exams, and eventually achieve their goals. By increasing education level among young generation, we can also develop our country as well.More specifically, this system is built to provide high quality education system to school students in regions, which gives them chance to learn/prepare/get motivation from one system.The software will allow communication between users and authors/admins via E-mail.
